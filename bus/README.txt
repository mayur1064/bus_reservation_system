This is a basic model of Bus Reservayion System in C++ !
Concepts Used :-

1) Object Oriented Concepts 
2) File Handling Concepts in C++
3) File Handling is done in .csv files in excel file 
4) Two excel files are created one containing list of all buses installed , and a excel file for each bus 
   information of seats reserved for passengers.
5) Functionalities such as login page , Install bus, Reserve Bus, Show Bus, Delete Bus,Cancel
   Reservation etc was main focus to be included.
6) IDE used was Codeblocks 17.12 for development.
7) Any other contributions to this project are welcomed :):)
